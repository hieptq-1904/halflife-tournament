# Half-Life Tournament Landing Page

Cinematic military gaming landing page inspired by modern FPS hero sections.

## Quick Start

Open `index.html` in a browser, or serve locally:

```bash
python3 -m http.server 8080
```

Then visit http://localhost:8080

## Structure

- `index.html` — Main landing page
- `css/style.css` — Styles and animations
- `js/main.js` — Gallery background switcher, mobile nav

## Customize

- Replace Unsplash background URLs in `index.html` and `js/main.js` with your own hero images
- Update tournament data, stats, and copy in `index.html`
- Adjust colors in `:root` variables in `css/style.css`
